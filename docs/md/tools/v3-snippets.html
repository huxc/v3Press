v3-snippets ​

v3-snippets 是一个适配 Vue3 模版、路由、Api 和 ES6 常用函数的 vscode 插件

安装 ​

为了安装扩展，需要启动命令(Ctrl + Shift + P 或 Cmd + Shift + P)并键入 Extensions。您可以选择显示已经安装的代码段或安装新的代码段。搜索 v3-snippets并安装。

支持的语言(file extensions) ​

  • JavaScript (.js)
  • Vue (.vue)
